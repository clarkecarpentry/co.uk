# Second Fix Carpentry

**URL:** /second-fix

## Description

Clarke Carpentry Contractors Ltd implements all aspects of second fix carpentry. Working in both commercial and domestic areas. This includes:

- Skirtings
- Architraves
- Traditional Hung Doors
- Door Sets
- Door Furniture (incl handles, locks, stops)
- Wooden Floor Installation
- Balustrading and Handrails
- Shelving
- Window Boards
- Staircases

If you require another service not listed then please get in touch.
