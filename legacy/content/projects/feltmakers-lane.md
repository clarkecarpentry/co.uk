# Feltmakers Lane (Hatterswood)

**URL:** /hatterswood
**Type:** New Build Development
**Client:** Juniper Homes
**Location:** Rangeworthy

## Description

Working with Juniper Homes, Clarke Carpentry Contractors Ltd were awarded the contract for this new build development in Rangeworthy. Works included joists, truss roof installation, stud walls, staircase, hardwood flooring, skirting, architraves and door sets.

**Work Completed:** February 2020
