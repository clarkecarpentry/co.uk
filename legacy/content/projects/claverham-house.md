# Claverham House

**URL:** /claverham-house
**Type:** Extension - Grade 2 Listed
**Partner:** CTS Design

## Description

This bespoke extension for a Grade 2 listed manor house was a collaborative project with CTS Design creating a new kitchen area and utility room. Clarke Carpentry Contractors Ltd erected all of the structural timber elements (walls and roof), dry lining walls and installation of traditional doors which included a bespoke external back door.

**Work Completed:** April 2018
