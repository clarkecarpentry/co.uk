# Rockview

**URL:** /rockview-1
**Type:** Domestic

## Description

This was a domestic project to improve an existing extension. We fitted new fascias, soffits, bargeboards and guttering to the outside of the extension. Internal works included fitting new window boards, doors, architraves and skirting. We also fitted a utility room with kitchen units.

**Work completed:** December 2019
