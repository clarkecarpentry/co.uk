# Sherwood Road (Hillside Court)

**URL:** /hillside-court
**Type:** Domestic Loft Conversion
**Partner:** By Design

## Description

A domestic loft conversion in collaboration with By Design. The client wanted to add an extra bedroom with ensuite to their house.

Work involved:
- Installation of RSJ steels
- Cut roof including flat roof above dormer window
- Moving the original staircase
- Install new staircase
- Skirting
- Architraves
- Hang doors

**Work Completed:** July 2020
