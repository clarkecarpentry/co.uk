# Walcot House

**URL:** /walcot-house
**Type:** Commercial Refurbishment
**Location:** Bath

## Description

Clarke Carpentry Contractors Ltd was awarded the contract to completely refurbish Walcot House (Bath), originally the site of an old nightclub. Managing the project to achieve a complete refit to a high end finish. This included project management of all the carpentry, dry lining, plastering, painting and bespoke joinery including new custom made bars. Running to a strict opening deadline, the project was completed on time.

**Work completed:** December 2017
