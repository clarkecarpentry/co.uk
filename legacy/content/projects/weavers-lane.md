# Weavers Lane (Tanhouse Lane)

**URL:** /tanhouse-lane
**Type:** New Build Development
**Client:** Juniper Homes
**Location:** Outskirts of Bristol

## Description

Working with Juniper Homes, Clarke Carpentry Contractors Ltd won the contract for all 1st and 2nd/Final Fix work on this development on the outskirts of Bristol. Work has included joists, truss roof installation, stud walls, staircase installation, hardwood flooring, windowboards, skirting, architraves and door sets.

**Work Completed:** September 2020
